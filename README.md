# Google Advanced Data Analytics - TikTok Project

This project was completed as part of the **Google Advanced Data Analytics Certificate** and focuses on analysing TikTok user engagement data to uncover trends, inform content strategy, and provide actionable recommendations for growth.

---

## Project Overview
TikTok has rapidly become one of the most influential social media platforms, and understanding its engagement patterns is key to optimising content performance.  
In this project, I worked with simulated TikTok engagement datasets to:
- Explore audience behaviour
- Identify high-performing content categories
- Provide data-backed suggestions for content optimisation

---

## Objectives
- Perform **exploratory data analysis (EDA)** to understand audience engagement metrics.
- Identify correlations between **content type, engagement rates, and posting frequency**.
- Create **data visualisations** to communicate trends effectively.
- Provide **recommendations** for maximising engagement.
